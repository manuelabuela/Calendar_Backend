Funcionalidades:
•	Criar notas;
•	Atualizar notas;
•	Apagar notas;
•	Mandar notificações sobre alterações sobre as notas;
•	Permitir Upload de files;
•	Apagar File;
•	Criar, atualizar, apagar Categoria (Curso, cadeira, Teste…);

*Funcionalidades como o crud de notas, categorias…são funções apenas reservadas para administradores. 
Existem dois tipos de admin, geral(eu ou a pessoa que estará a gerir a plataforma de forma geral) e 
os reesposáveis de cada turma, que terão algumas permissões reduzidas.
